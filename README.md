# Password Generator Refactoring Project

Projeto acadêmico demonstrando a refatoração de um gerador de senhas, aplicando:
- Princípios SOLID
- Padrões de Design
- Clean Code
- Testes unitários

## Estrutura
- `legacy/`: Código original
- `src/`: Código refatorado
- `tests/`: Testes unitários

## Pré-requisitos
Python 3.8+
